# Proyecto-Sistema-de-Gestion-de-Cocina-Economica

🍽️ Sistema de Gestión de Cocina Económica
📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un sistema para la gestión de una cocina económica, cuyo objetivo es optimizar procesos como la administración de pedidos, ventas, inventario, clientes y reportes.

El sistema fue diseñado aplicando principios de Ingeniería de Software, modelado UML y buenas prácticas de diseño.

🎯 Objetivo

Desarrollar una solución que permita:

Mejorar la eficiencia operativa
Automatizar procesos clave
Facilitar la toma de decisiones mediante reportes
